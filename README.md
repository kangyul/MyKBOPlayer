# MyKBO Player

내가 관심 있는 KBO 선수의 기록, 최신 뉴스, 커뮤니티 반응을 한 곳에서 확인할 수 있는 웹앱입니다!

## Description

* 관심 있는 선수를 최대 3명까지 지정할 수 있습니다.
* 내가 선택한 선수의 이번 시즌 기록, 최신 뉴스, 그리고 선수에 관한 여러 커뮤니티 글들을 바로 확인할 수 있습니다.
